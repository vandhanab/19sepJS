Data Types

Data types are divided into two types

1.Primitive data types
2.Non-Primitive data type

Primitive data type :

  • Primitive data types are the fundamental building blocks used to represent single values.
  • Primitive data type are directly stored in memory & are immutable.
    • They are:
    1. Numbers:The Numbers represents numeric values,including integers & floating-points Numbers.
    2. String:It represents textual data,enclosed in single or double quotes.
    3. Boolean:It represents a logical value,True or False.
    4. Null:It represents an intentional absence of any object value.
    5. Undefined:It represents a variable that has been declared but has not been assigned a value.

Non-Primitive data type

  • Non-Primitive data types are Mutable, Meaning there values can be changed after they are created, and they are stored and accessed by reference rather than by value.
    • They are:
    1. Object: It represents a collection of key-value pairs, where keys are strings & values can be any data types , including other objects.
    2. Array:It represents an orders collection of elements,accessed by index,starting from zero.
